About This Project

Objective

The AI Recommendation System is designed to demonstrate how machine learning can be integrated into a web application to provide personalized user experiences. It analyzes user preferences and matches them with items using Natural Language Processing and Cosine Similarity.

AI Recommendation Process

1. Data Ingestion: Item descriptions and categories are processed from a dataset.
2. Vectorization: Text data is converted into numerical vectors using TF-IDF (Term Frequency - Inverse Document Frequency).
3. Cosine Similarity: The user's input is vectorized and compared against all items mathematically to find the closest match.

Technologies Used

  • Python 3
  • Flask (Web Framework)
  • Scikit-learn (ML Library)
  • Pandas & NumPy
  • Bootstrap 5
  • Chart.js

Team Details

Developed as a professional AI integration project. Designed for performance, accuracy, and a clean user experience.
